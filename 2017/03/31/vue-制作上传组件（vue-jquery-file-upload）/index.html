vue 制作上传组件（vue + jquery-file-upload）

发表于 2017-03-31

背景

最近接手的一个项目（项目的前端技术选型是 webpack + vue）有大量的文件（文本、图片、视频、录音），需要上传到七牛。于是迫切需要一个上传组件。

上传组件要满足一下几个要求：

  1. 能指定不同文件类型，过滤不符合的文件类型。

  2. 支持批量上传。

  3. 自带 BGM 进度条提醒。

  4. 上传后预览功能。

  5. 兹瓷西方和其他国家多款浏览器。

思前想后最后决定是用上 jQuery-file-upload。
